# 健康診断

たいていの場合、健康診断は会社の都合で日程が決まる（と思う）。
2019年は年始休み明け初日に2018年度分の健康診断を行った。
2019年度分は12月だったので、1年間に2回健康診断を受けたことになる。
1月に受けた方ではバリウムを飲んだが、12月に受けた方は胃カメラだった。
意識がある状態での胃カメラは初めてだったが、まあなんとかなった。
食道、胃、十二指腸は特に大きな問題がなかったので良かった。

とはいえ、トータルではB判定で、要経過観察の項目があった（例年通り）。
